功能概述

12CM 是藍新金流的收款方（Merchant），須完成藍新金流帳號申請與綁定，才能在 OA Plus 上支援付費 Plugin。

金流操作（授權、請款、退款、自動續扣、retry）全部由 OA Plus 平台代為執行，12CM 不直接呼叫藍新 API。12CM 的角色是透過 Webhook 與 Open API 取得付款結果，並據此處理內部權益開通與發票流程。

ℹ️
定位說明 本文件涵蓋：金流整合責任範圍劃分、前置條件清單、付款流程架構、Trial 差異、自動續扣、扣款失敗寬限期機制，以及 12CM 的實作項目。

責任範圍劃分 內部邏輯

三方角色的責任邊界如下：

LINE OA Plus 平台
  • 建立 Subscription
  • 建立 Payment
  • 導向藍新付款頁
  • 接收藍新付款結果
  • 管理自動續扣時序
  • 執行 retry / 寬限期
  • 更新 Subscription 狀態
  • 發送 Webhook 通知 12CM
藍新金流
  • 信用卡授權
  • 請款 / 補扣
  • 退款處理
  • 回傳付款結果給 LINE
12CM（Partner）
  • 申請藍新帳號（收款方）
  • 綁定帳號至 OA Plus
  • 接收 Webhook 通知
  • 查詢付款結果（Open API）
  • 判斷是否開票
  • 判斷是否啟用進階版
✅
12CM 不直接呼叫藍新 API 所有藍新金流操作均由 LINE 平台代為處理。12CM 只需完成帳號申請與綁定，後續僅需處理 LINE 發出的 Webhook 事件。

Trial（試用）訂閱的付款差異 LINE 規格

情境付款行為藍新動作
無 Trial 一般訂閱 立即向使用者請款 授權 + 立即請款
含 Trial 訂閱 僅進行付款方式授權，不立即扣款；Trial 到期後建立第一期帳單並正式扣款 授權綁卡（不請款）→ Trial 到期後請款
ℹ️
Trial 期間為 1 個月（30 天），由 LINE 依確認天數設定。授權→請款的時序均由 LINE 平台自動管理，12CM 無需額外處理。

自動續扣機制 LINE 規格

  • Subscription 有效且到達當期結束點時，平台自動發起續扣，每月同一天執行
  • 12CM 無需主動觸發，只需監聽後續 Webhook
續扣結果平台行為12CM 收到
✅ 成功 Subscription 維持有效，建立下一期計費紀錄 PAYMENT_SUCCEEDED
❌ 失敗 進入寬限期（Grace Period），Subscription 主狀態維持 ACTIVATED （無立即通知，等 retry 結果）

付款方式更新 LINE 規格

  • 有效訂閱期間，Admin 角色可更新付款方式（綁定新卡）
  • 更新後僅影響未來續扣或補扣，不影響已完成的歷史扣款
  • 12CM 系統無需針對付款方式更新做任何處理，此行為完全由 LINE 平台管理

12CM 需實作項目 內部邏輯

①
接收 PAYMENT_SUCCEEDED Webhook

驗證 X-DX-Signature，解析 payload，取得 paymentOrderId

②
呼叫 Retrieve Payment Open API

取得付款快照，確認付款結果，作為後續開票依據（詳見 Spec 01）

③
接收 ACTIVATION_CHANGED Webhook

訂閱啟用或失效（EXPIRED）時，同步更新 12CM 系統中的服務權益（詳見 Spec 03）

④
訂閱狀態同步

根據 Subscription 狀態與 plugin level 判斷商戶是否具備付費功能存取權（詳見 Spec 03 §6）

ℹ️
Webhook 僅為事件通知，不含完整明細。收到 Webhook 後，必須再呼叫對應 Open API 查詢完整資訊後再處理。

Webhook 重試規範 LINE 規格

OA Plus 對 5xx / 429 回應會自動重試：

Response StatusOA Plus 行為
2xxSuccess，不重試
4xxPermanent failure，不重試
5xx 或 429Transient failure，retry 最多 3 次（1s → 2s → 4s），全部失敗後標記為 failed

注意事項

  • 禁止在 Staging / Production 環境進行壓力測試
  • Open API 僅允許 IPv4 CIDR 格式的白名單 IP 存取，需提前向 LINE 登記
  • Auth Token 有效期為 5 分鐘，需在 token 過期前重新取得
  • Reverse Proxy 不支援 3xx redirect（除 304 Not Modified），後端不可回傳 redirect 回應
